The <time> Element Showcase

Examples of the <time> Element

Date Only (YYYY-MM-DD)

The following is a date in the format YYYY-MM-DD:

January 1, 2023

Time Only (24-Hour Format)

The following is a time in the 24-hour format:

13:45

Date and Time Combination

The following is a date and time combination:

January 1, 2023, at 13:45

Datetime with Timezone

The following is a date and time with timezone information (UTC):

January 1, 2023, at 13:45 UTC

Human-readable Date with Datetime Attribute

The following is a human-readable date with a datetime attribute:

New Year's Day 2023
